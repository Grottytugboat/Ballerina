# Client Logos

Place the following logo files in this directory:

1. **rolls-royce.png** - Rolls Royce logo (transparent PNG, recommended size: 200-300px width)
2. **jaguar.png** - Jaguar logo (transparent PNG, recommended size: 200-300px width)
3. **formula1.png** - Formula 1 logo (transparent PNG, recommended size: 200-300px width)
4. **pharmaceutical.png** - Pharmaceutical companies logo or a generic pharma logo (transparent PNG, recommended size: 200-300px width)

## Logo Sources

### Recommended Sources (Free/Legal):

1. **Wikimedia Commons** (Public Domain/CC Licensed):
   - Rolls Royce: https://commons.wikimedia.org/wiki/File:Rolls_royce_motorcars_logo.svg
   - Formula 1: https://commons.wikimedia.org/wiki/File:Formula_1_logo.svg
   - Jaguar: Search Wikimedia Commons for "Jaguar logo"

2. **Brands of the World**: https://www.brandsoftheworld.com
   - Search for each brand name
   - Download SVG or PNG formats

3. **Official Brand Guidelines**:
   - Check each brand's official website for logo usage guidelines
   - Some brands provide logo downloads for media/partners

### For Pharmaceutical Companies:

Since "Pharmaceutical Companies" is generic, you can either:
- Use a generic medical/pharma icon
- Use a combination of 2-3 major pharma company logos (Pfizer, J&J, etc.)
- Use text only for this one

## Format Requirements

- **Format**: PNG with transparent background (or SVG converted to PNG)
- **Size**: 200-300px width (height will scale proportionally)
- **Style**: Logos will be displayed in grayscale by default, hover shows full color
- **Optimization**: Compress images for web use (use tools like TinyPNG)

## How to Add Logos

1. Download logos from the sources above
2. Ensure they have transparent backgrounds
3. Resize to approximately 200-300px width
4. Save with the exact filenames listed above
5. Place in this directory: `public/images/logos/`

## Note

If logo files are not found, the component will automatically fall back to displaying the client name as text, so the site will work even without logos initially.
